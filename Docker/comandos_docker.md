# Comando para instalação do Docker.
 - sudo apt update
sudo apt install docker.io docker-compose -y

- caso não queira reinicar o computador usando o comando - sudo apt install docket.iol docker-compose - ele já fica em ambiente de teste. 

- tem algums informações sobre o Docker no comando - docker --help - menu de ajuda onde mostra os principais comandos.

- docker pull wordpress - comando onde vai baixar as imagens para ser usado no Docker.
- docker images - mostra as imagens disponiveis.
- docker pull ubuntu - baixar o sistema mais recente da versão da canônico
- docker ps - lista os conteiners em uso
