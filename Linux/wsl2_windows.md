## Instalão do WSL2
### Comandos
- wsl --install - instação, precisa reiniciar para testar.
- wsl -l -v - verificar a versão que foi instalada

## Instalão do Ubuntu
- Baixado direto pelo Microsof Store

## Instalção Sudo
- sudo apt update - Instalação do sudo
- sudo apt update && sudo apt upgrade -y - Comando para garantir que ele está pronto para uso.
