## O que é Linux? 
- Linux é um sistema operacional, tecnicamente o núcleo (Kernel) do sistema. Usado nos sevidores, ele é usado como base para criar outros sistema operacionais.

## Porque ele é importante?
- Porque ele está como base em grande parte das tecnologias criadas.

## Quem faz o Linux?
- Todas as empressas podem criar ele, com finalizades diferentes e aumentando a qualidade do Kernel e a versatilidade do ele pode fazer de acordo com a necessecidade.

## Quem Usa?
- Todos podem usar, fazendo o Linux de acordo com o sistema que precisa.

## Porque provavelmente ele não é o que voce pensa?
- Todos pensam que ele é apenas um sistema operacional, porém ele vai além disso, alem de gerenciar hardware ele é parte mais importante do sistema operacional como o (Kernel), a parte mais importante para fazer o sistema operacional funcionar.

## Para que ele Serve?
- Como uma familia de sistema operacional, ele serve para tudo que é sistema operacional como base para fazer funcionar.
